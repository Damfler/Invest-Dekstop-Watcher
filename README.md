# T-Bank Invest — виджет в трее v6+v7

## Что нового

### v6 — Стабильность и архитектура

**Исправление заморозки меню (главный баг)**
Меню теперь строится в момент открытия, а не при обновлении данных.
Фоновый поток API не касается меню — заморозка устранена полностью.

**Модульная структура** — 9 файлов вместо одного на 960 строк:
```
main.py          — точка входа, логирование
config.py        — загрузка конфига, слияние дефолтов
api.py           — T-Bank API с retry
cache.py         — дисковый кэш
utils.py         — форматирование
icons_gen.py     — генерация иконок
autostart.py     — реестр Windows
notifications.py — toast-уведомления
analytics.py     — YTM, аллокация, топ-муверы
data_store.py    — хранилище данных
menu.py          — построитель меню
app.py           — главный класс
```

**Retry при ошибках API**
3 попытки с задержкой 2→4→8 сек при 5xx / таймауте.
401/403 — сразу ошибка без retry.

**Лог ошибок в файл** `tbank_errors.log`
Ротация: макс 1 МБ, хранится 3 файла.

**Кэш данных на диск** `cache.json`
При старте — мгновенно показывает последние данные.
Кэш сохраняется при выходе. Максимальный возраст — 24 ч.

**Sandbox-режим**
`"use_sandbox": true` в config.json → работа с тестовым API без реального портфеля.

### v7 — Новые данные

**Купон без суммы → НКД × кол-во**
Если API не вернул `payOneBond` — используем НКД как оценку.
Пометка `≈` показывает что значение расчётное.

**Топ-3 позиции: рост / падение за день**
Секция «📈 Лучшие за день» / «📉 Худшие за день» в меню.

**Аллокация портфеля**
`🗂 Аллокация: Акции 45% · Облигации 38% · Фонды 12% · Валюта 5%`

**Средний YTM облигаций**
Взвешенная доходность к погашению по всем облигациям в портфеле.

**Горизонт событий из меню**
30 / 60 / 90 дней — переключается кликом, сохраняется в config.json.

**Сортировка событий облигаций**
По дате (дефолт) или по сумме выплаты — переключается в меню.

**Настраиваемые пороги уведомлений**
- `notify_move_pct` — % движения портфеля для уведомления (дефолт 1.0)
- `notify_offer_days` — за сколько дней предупреждать об оферте (дефолт 2)

## Установка

```bash
pip install -r requirements.txt
```

Получить токен: Т-Инвестиции → Настройки → Открыть API → только чтение.

```bash
python main.py
```

При первом запуске создаётся `config.json` — вставьте токен и перезапустите.

## Автозапуск

Кнопка «Автозапуск» прямо в меню — включает/выключает запись в реестр Windows.
Запускается через `pythonw.exe` — без окна консоли.

---

## Sandbox-режим для разработки

```json
"use_sandbox": true
```

Переключает API на `sandbox-invest-public-api.tinkoff.ru`.
Безопасно разрабатывать и тестировать без реального портфеля.
